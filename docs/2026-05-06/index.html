oh-my-free-models - 무료 LLM 중 지금 가장 빠른 모델로 코딩 에이전트를 라우팅하는 로컬 프록시

oh-my-free-models(omfm)는 여러 무료 LLM provider 중 현재 가장 빠른 모델로 요청을 자동 라우팅하는 로컬 프록시임. OpenAI 또는 Anthropic 호환 에이전트의 baseURL을 localhost로 변경하고 무료 모델을 설정하면 동작함. latency, rate-limit, quota 변동에도 자동으로 최적 모델을 선택해 요청을 안정적으로 처리함.

Cursor, Continue 등 코딩 에이전트에서 무료 모델 여러 개를 로테이션하며 rate-limit 없이 안정적으로 사용할 때 로컬 프록시로 설정.
