신경망과 암호화 알고리즘은 왜 이토록 유사한가?

신경망과 대칭 암호(AES, ChaCha20 등)는 서로 독립적으로 거의 동일한 알고리즘 구조에 도달했다는 점을 분석한 글이다. 선형 변환과 비선형 변환을 반복 적용하고, 행-열 단위로 상태를 혼합하는 구조가 Transformer의 어텐션/FFN 층과 AES의 ShiftRows/MixColumns에서 동일하게 나타난다. 두 분야 모두 약한 정확성 조건(역변환 가능성 vs 미분 가능성), 철저한 정보 혼합 필요성, 하드웨어 효율 우선이라는 세 가지 제약이 수렴 진화를 이끌었다고 설명한다.

암호학의 substitution-permutation 구조를 참고해 새로운 신경망 아키텍처(예: RevNet류)를 설계하거나, 역으로 신경망 기법을 경량 암호 설계에 적용하는 크로스도메인 연구에 활용할 수 있다.

우버, 2026년 AI 예산을 클로드 코드에 4개월 만에 소진

우버가 2026년 연간 AI 예산을 Claude Code와 Cursor 등 AI 코딩 도구에 4개월 만에 전부 소진했다. 엔지니어 1인당 월 500~2,000달러의 API 비용이 발생하며, 엔지니어 95%가 AI 도구를 월간 사용 중이다. 커밋된 코드의 70%가 AI 보조 개발에서 생성되었으며, Claude Code 사용량은 2025년 12월~2026년 2월 사이 2배로 증가했다. CTO는 AI 예산 편성을 근본적으로 재검토해야 한다고 밝혔으며, 연간 R&D 지출 34억 달러 규모에서 비용 지속 가능성이 과제로 떠올랐다.

대규모 조직에서 AI 코딩 도구 도입 시 엔지니어당 월 500~2,000달러 수준의 비용을 기준으로 예산을 산정하고, 사용량 급증에 대비한 분기별 예산 리밸런싱 체계를 마련할 수 있다.

우분투 서버, '지속적인 국경 간 공격'으로 오프라인 전환

AI의 물 사용량, 대중이 생각하는 것보다 적다

음악 플랫폼을 운영하거나 콘텐츠 마켓플레이스를 구축할 때, AI 생성 여부를 표시하는 인증 체계를 설계하는 참고 사례로 활용할 수 있다.
