거실의 스마트 TV가 AI 스크래핑 경제의 노드가 되다

Bright Data 등 거주용 프록시 회사가 스마트 TV 앱에 SDK를 심어 AI 학습용 웹 스크래핑 네트워크의 종단점으로 변환. 스마트 TV는 항상 전원 상태와 안정적 WiFi로 휴대폰보다 우수한 프록시 역할. SDK는 기기 텔레메트리를 수집하며 VPN을 우회하고 모니터링 도구를 회피하는 두 가지 검사 우회 기법 사용. 사용자가 유휴 상태로 간주해도 실제로는 지속적으로 네트워크 자원 소비.

네트워크 관리자는 proxyjs.brdtnet.com, clientsdk.bright-sdk.com 도메인 차단 및 TLS 인증서 모니터링으로 무단 프록시 노드 생성 방지 가능.
